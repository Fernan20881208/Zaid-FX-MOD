# Zaid-FX-MOD v0.4.0 — ZaidLux

This release combines the complete ZaidLux visual pipeline, synchronized presets, color-preserving highlight correction and the first internal Android screen recorder.

## Highlights

- Two-pass post-processing with Low, Medium, High and Ultra quality levels
- Bloom, emissive lighting, ambient occlusion, reflections and light rays
- Fake HDR, full color grading, local contrast, specular highlights, depth separation and sharpen
- Reactive lighting for gameplay events
- Default, Glow, ZaidLux, ZaidLux Neon, Cinematic, Cyberpunk and ZaidLux Performance presets
- Persistent sliders, toggles, quality and custom preset values
- Corrected magenta/yellow clipping and restrained high-intensity color output
- Internal H.264/MP4 recorder with Record, Stop, Save and Delete flow
- Current project-tree privacy and maintenance cleanup

## Recorder limitation

The initial internal recorder captures video only. Game audio is not included. Saved MP4 files are placed in the mod save directory under `recordings`.

## Compatibility

Built for Geometry Dash 2.2081, Geode 5.7.1 and Android64. Visual behavior, recorder output and performance still require validation on real devices and multiple GPU families.
